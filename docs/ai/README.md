# AI / Agents — Índice

Arquivos de skills e referências de agentes estão localizados em `.agents/skills/` e são usados por ferramentas/assistentes automatizados. Não mova arquivos dessa pasta sem confirmar, porque essas ferramentas podem depender dos paths atuais.

## Principais entradas

- `.agents/skills/supabase/SKILL.md`
- `.agents/skills/supabase-postgres-best-practices/SKILL.md`

## Uso recomendado

- Consulte este README antes de editar conteúdos em `.agents/`.
- Para documentação do time, prefira criar notas em `docs/ai/` em vez de editar os arquivos dentro de `.agents/`.
